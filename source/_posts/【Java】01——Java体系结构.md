&emsp;&emsp;Java是一个开发平台，为开发者提供了一套完整的“生态系统”。通过Java程序设计语言设计与平台无关的应用，不需要考虑底层与操作系统or硬件的交互。应用会被虚拟机编译为class规范文件，供虚拟机加载运行。同时Java提供了应用编程接口，即常用的编程基础库，便利开发者的应用实现。
